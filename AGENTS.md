# Repository-Instruktionen

- Git Bash fuer Shell-basierte Tools bevorzugen, wenn es der Aufgabe hilft.
- Aenderungen fokussiert halten und unnoetige externe Referenzen vermeiden, wenn sie nicht angefordert sind.
- Saemtliche Inhalte fuer Labs muessen in deutscher Sprache gehalten werden. Etablierte Fachbegriffe wie agentic programming, workflow, prompt oder MCP duerfen bestehen bleiben, aber erklaerender Inhalt und Aufgabenstellungen bleiben auf Deutsch.
